### About

This project is a manager tasks using React and Rails.

### Prerequesites

Install server

```
  $ cd path/cathfy-api/
  $ rails s -p 3000
```

### Getting Starting

```
 $ git clone https://github.com/acmesquita/cathfy-react.git
 $ cd cathfy-react/
 $ npm install
 $ npm start
```
