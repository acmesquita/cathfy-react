### Prerequesites

Install server

```
  $ cd path/cathfy-api/
  $ rails s -p 3000
```

### Getting Starting

```
 $ git clone https://github.com/acmesquita/cathfy-react.git
 $ cd cathfy-react/
 $ npm install
 $ npm start
```
